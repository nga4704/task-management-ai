# PROJECT CONTEXT

## Project Name

AI-Powered Project Management System

Tiếng Việt:
Xây dựng hệ thống quản lý dự án thông minh tích hợp trí tuệ nhân tạo

## Objective

Xây dựng hệ thống quản lý dự án hiện đại hỗ trợ:

* Quản lý Team
* Quản lý Project
* Quản lý Task
* Theo dõi tiến độ
* Phân tích hiệu suất
* Hỗ trợ ra quyết định bằng AI
